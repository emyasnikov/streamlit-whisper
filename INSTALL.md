# Installation

## Prerequisites

### Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

#### Python

```bash
brew install python3.12
```

## Python

### Environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### Packages

```bash
pip install -r requirements.txt
```
